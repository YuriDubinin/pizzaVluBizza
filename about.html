This is a project of a fictional pizzeria. It is my first project where I apply my HTML / CSS skills. The project is written in pure HTML and CSS(JS was used only for the burger-menu). All CSS is written in the SASS preprocessor with SCSS syntax.The main goals of this project were to build markup using the Flexbox module, using the BEM methodology, as well as using the adaptive one, so that the project is displayed qualitatively on the displays of various devices (using media queries). This is my first experience of this type of work. That's all, thank you for your attention.
